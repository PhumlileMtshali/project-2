# 🍽️ Recipe Rhapsody

## 📌 Overview

Recipe Rhapsody is a recipe website developed using **HTML and CSS**. The platform provides a collection of recipes categorized into different dish courses, allowing users to easily explore and discover meals for various occasions.

The project focuses on creating a clean, structured, and visually appealing layout for browsing recipes.

---

## 🛠️ Technologies Used

* HTML5
* CSS3

---

## ✨ Features

* Categorized recipes (Starters, Main Courses, Desserts, and Drinks)
* Clear and user-friendly navigation
* Visually appealing and organized layout
* Structured recipe sections for easy reading
* Styled recipe cards and content sections

---

## 🍴 Recipe Categories

The website includes:

* Starters
* Main Courses
* Desserts
* Drinks

Each category presents recipes with well-structured content such as ingredients and preparation steps.

## 🔧 How to Run the Project

1. Download or clone the project files
2. Open the project folder
3. Open the `index.html` file in a web browser
